# 烟花研究所

一个可以模拟研究各种烟花效果的网站

它可以展示各种绚丽多彩的烟花效果 并让人仿佛置身于真实烟花的绚丽世界之中

```
祝大家新年快乐，蛇来运转，画蛇添福气！
```
